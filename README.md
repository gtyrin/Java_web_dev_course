# Java_web_dev_course

Задание к курсу "Java web development"
======================================
12.	Система Авиакомпания. Авиакомпания имеет список рейсов. Диспетчер формирует летную Бригаду (пилоты, штурман, радист, стюардессы) на Рейс. Администратор управляет списком рейсов.

Требования к выплнению проекта 
===============================
1. Информацию о предметной области хранить в БД (рекомендуется MySQL). 
2. На основе сущностей предметной области создать классы их описывающие. 
3. Реализовать функциональности, предложенные в постановке конкретной задачи. 
4. Классы и методы должны иметь отражающую их функциональность названия и должны быть грамотно структурированы по пакетам. 
5. При разработке использовать журналирование событий (Log4j или аналог). 
6. Код должен содержать комментарии.

Общее описание работы
=====================
В разрабатываемой системе работа проводится двумя типами пользователей. "Администратор" ведет НСИ и формирует список рейсов. "Диспетчер" формирует летные экипажи. Работа с системой начинается с диалога авторизации пользователя для обеспечения доступа к соответствующей подсистеме.


Описание таблиц схемы
---------------------
Графическое представление схемы можно увидеть в https://github.com/gtyrin/Java_web_dev_course/blob/master/doc/other/Flight_screw_schema.mwb

- airport            - справочник аэропортов
- crew               - таблица формируемых экипажей
- crew_man           - описание члена экипажа
- crew_spec          - справочник специализаций членов экипажа
- flight             - список сформированных рейсов
- operator           - справочник пользователей системы
- operator_type      - справочник типов пользователей системы
- plane              - справочник самолетов авиакомпании
- plane_model        - описание технических характеристик моделей самолетов авиакомпании
